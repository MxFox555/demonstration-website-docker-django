Prepare your data


Transform your data to a readable format for the API.


This can be done however you like, But the data needs to be in the following format.


Post your data


All API requests go through https://stockinterestmetric.com/api/request/ as a post request.


You can use curl to test requests. eg:



Requests can be made through Python scripts or other languages as long as they are encoded to bytes from a JSON format.


If your recieve an error, don't worry it does not subtract from your daily API calls.

Recieve your data


Your data will be sent back as a JSON file with the series of values, 'stable' and 'volatile'.

Your are now able to graph these ontop of your 'open', 'high', 'low' and 'close' values.


Graph your data


With your acquired data, you can now graph it.
